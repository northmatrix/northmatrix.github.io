Order

northmatrix

Overview

The Order room demonstrates weak XOR encryption using a repeatingg key. By leveraging a known plaintext prefix, the key can be extracted and the full message decrypted.

Solution

from itertools import cycle

order = b"ORDER:"
msg = "1c1c01041963730f31352a3a386e24356b3d32392b6f6b0d323c22243f63731a0d0c302d3b2b1a292a3a38282c2f222d2a112d282c31202d2d2e24352e60"
msg = bytes.fromhex(msg)
key = bytes(l ^ m for m, l in zip(msg, order))
print(f'Key: "{key.decode()}"')
plt = bytes(k ^ m for m, k in zip(msg, cycle(key)))
print(f'Plaintext: "{plt.decode()}"')
